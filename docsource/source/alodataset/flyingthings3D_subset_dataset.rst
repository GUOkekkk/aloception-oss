Flyingthings3D Subset
---------------------

This dataset is part of the `Scene Flow datasets
<https://lmb.informatik.uni-freiburg.de/resources/datasets/SceneFlowDatasets.en.html>`_
provided by the Computer Science Group of the University of Freiburg.

This is a synthetic dataset containing optical flow and disparity, with occlusion, for two cameras (left/right).


- For more information about the dataset, please refer to the `CVPR2016 paper <https://lmb.informatik.uni-freiburg.de/Publications/2016/MIFDB16/>`_.
- To download the dataset, please follow instructions at `this link <https://lmb.informatik.uni-freiburg.de/resources/datasets/SceneFlowDatasets.en.html>`_ in the section *DispNet/FlowNet2.0 dataset subsets*.
