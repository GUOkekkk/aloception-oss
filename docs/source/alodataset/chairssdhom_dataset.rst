ChairsSDHom
###########

This dataset is part of the `Optical Flow datasets
<https://lmb.informatik.uni-freiburg.de/resources/datasets/FlyingChairs.en.html>`_
provided by the Computer Science Group of the University of Freiburg.


- For more information about the dataset, please refer to the `CVPR2017 paper <https://lmb.informatik.uni-freiburg.de/Publications/2017/IMKDB17/>`_.
- To download the dataset, please follow instructions at `this link <https://lmb.informatik.uni-freiburg.de/resources/datasets/FlyingChairs.en.html>`_ in the section *The "ChairsSDHom" Dataset*.
